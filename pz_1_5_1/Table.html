МІЙ ЕЛЕКТРОННИЙ ДОВІДНИК HTML
Призначення Назва тегу Короткий опис Стандарт
Для каркасу документу <html> Початок і закінчення HTML-документа HTML5
<head> Eлемент-контейнер для метаданнuх HTML-документа
Для роботи з текстом <p> Параграфu y текстi
<b> Задає напівжирний нарис частини тексту, не надаючи акценту
або важності виділеному
<i> Виділяє частину тексту курсивом, не придаючи йому додаткового акценту
<u> Виділяє частину тексту підкресленням, без додаткового акценту
<blockquote> Виділяє текст як цитату, використовується для опису великих цитат
<br> Перенос тексту на новий рядок
<del> Виділяє текст як видалений, перекреслюючи його
Для гіперсилок <a> Створює гіпертекстові силки
