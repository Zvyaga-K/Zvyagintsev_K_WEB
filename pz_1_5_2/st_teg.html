Назва тегу <h3>

HTML тег <h3> (h з англ. слова heading - заголовок) використовується для визначення заголовків на веб-сторінці. Має визначене значення (важність), яка визначається цифрою після букви h, таким чином <h1> — має найбільшу важність, а <h6> - найменшу

Синтаксис

<h3> Текст заголовку третього рівня </h3>

Атрибути

align Вирівнювання заголовка тексту по краю
id Вказує ім'я стилевого ідентифікатора
class Визначає ім'я класу, яке дозволяє пов'язати тег із стилевим оформленням
dir Задает направление и отображение текста — слева направо или справа налево

Браузери

Internet Explorer Chrome Opera Safari Firefox Android Apple
3.0+ 1.0+ 2.0+ 1.0+ 1.0+ 1.0+ 1.0+

Приклад:


<!DOCTYPE html>
<html>
<head>
	<titel>документ</titel>	
</head>
<body>
	<h1>Текст заголовку першого рівня</h1>
	<h3>Текст заголовку третьго рівня</h3>
	<h6>Текст заголовку шостого рівня</h6>
</body>
</html>
