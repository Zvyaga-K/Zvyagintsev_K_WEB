Назва атрибуту: xmlns

Використовується у XHTML для оголошення простору імен — сукупності імен тегів та атрибутів призначених для збереження унікальності імен. Значенням атрибута xmlns є адреса файлу, якй оголошує необхідний простір імен, для XHTML значення завжди дорівнює http://www.w3.org/1999/xhtml. Цей атрибут також є частиною синтаксису XML, тому може використовуватись для додавання префіксу, щоб оголосити додатковий простір імен для документу. Наприклад: xmlns:my задает адреса файлу для елементів із префіксом my.

Синтаксис:

HTML <html>
XHTML <html xmlns="www.w3.org/1999/xhtml">

Значення:

У якості значення вказується повна адреса документа з описом простору імен. Локальний шлях та пустий рядок недопустимий. Для XHTML значення завжди http://www.w3.org/1999/xhtml.

  "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
 <head>
  <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
  <title>Тег HTML, атрибут xmlnsКорректный XHTML-документ.</p>
 </body>
</html>
